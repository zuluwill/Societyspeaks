## 📚 Best Practices to Prevent Future N+1s

### 1. **Avoid Query-Executing Properties in Models**
```python
# ❌ BAD - triggers query every access
@property
def item_count(self):
    return self.items.count()

# ✅ GOOD - check for cached value first
@property
def item_count(self):
    if hasattr(self, '_cached_item_count'):
        return self._cached_item_count
    return self.items.count()
```

### 2. **Batch Fetch Counts in List Routes**
```python
# In routes that display lists:
brief_ids = [b.id for b in briefs]
counts = db.session.query(
    BriefItem.brief_id,
    func.count(BriefItem.id)
).filter(BriefItem.brief_id.in_(brief_ids)).group_by(BriefItem.brief_id).all()

for brief in briefs:
    brief._cached_item_count = counts.get(brief.id, 0)
```

### 3. **Use `joinedload` for Relationships Accessed in Templates**
```python
# If template does {{ item.user.name }}
query = Model.query.options(joinedload(Model.user)).all()
```

### 4. **Test with SQL Logging Enabled**
```python
# In development config:
SQLALCHEMY_ECHO = True
# Or:
import logging
logging.getLogger('sqlalchemy.engine').setLevel(logging.INFO)
```
