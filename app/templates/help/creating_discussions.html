Creating Discussions

Learn how to create and configure effective discussions using Pol.is

Step-by-Step Guide

1. Create a Pol.is Discussion

  • Click the "Create Discussion" button
  • Sign in or create a Pol.is account
  • Configure your discussion settings

2. Add Seed Comments

  • Add at least 10-25 initial comments
  • Use AI tools to help generate diverse viewpoints
  • Learn more about seed comments

3. Configure Settings

  • Set geographic scope (Global, Country, or City)
  • Choose relevant topics
  • Add description and context

Discussion Best Practices

Creating an effective discussion requires careful thought and preparation:

  • Choose a clear, specific topic
  • Write a detailed description that provides context
  • Add diverse seed comments to encourage participation
  • Consider your target audience when setting geographic scope
  • Share your discussion on relevant platforms and communities

Using Pol.is Effectively

Key features of Pol.is:

  • Real-time visualization of opinions
  • Automated clustering of viewpoints
  • Support for multiple languages
  • Moderation tools
