    📊 {{ _('Methodology') }}

    {{ _('Clustering pipeline.') }} {{ _('Build a participant × statement vote matrix (values in {-1, 0, +1}). Missing votes are imputed with the statement mean so that sparse voters do not skew the principal components. PCA (2 components) followed by sparsity-aware scaling — participants who voted less are stretched away from the centroid by √(N_total/N_voted) — then %(method)s clustering with cosine distance. k is chosen by silhouette score over k ∈ [2, 10].', method=analysis.method.title()) }}

    {{ _('Consensus.') }} {{ _('A statement is surfaced only if the Wilson 95%% score interval on overall agreement clears 70%%, AND every opinion group’s own Wilson lower bound clears 60%%. Raw percentages alone are never sufficient — 7/10 agreement is not "70%% consensus" when its confidence interval starts at 40%%.') }}

    {{ _('Representativeness (what each group believes).') }} {% set _fdr = analysis_meta.get('fdr_method', 'bh')|upper %} {{ _('For every (group, statement) pair we compute the lift — P(agree | in-group) / P(agree | out-of-group) — run a one-sided Fisher’s exact test on the 2×2 contingency table, and apply %(fdr)s FDR correction across all tests at q=0.05. Only FDR-significant statements receive the ✓ badge; everything else is marked "suggestive only".', fdr=_fdr) }}

    {{ _('Bridges and divisions.') }} {{ _('Bridges require mean cross-group agreement ≥65%% (or shared rejection ≤35%%) with inter-group variance ≤0.15. Divisive statements use a pre-specified omnibus test: permutation χ² across ALL clusters on each statement, with the fraction of shuffled assignments as extreme as observed reported as the p-value. FDR is applied to those omnibus p-values. Gap CIs use Newcombe’s MOVER-Wilson (1998) Method 10. The pairwise Fisher p on the extreme pair is surfaced as a small-sample guard on the displayed gap, not a simultaneous test.') }}

    {{ _('Reproducibility.') }} {% if analysis_meta.get('stability_runs') %} {% set _ari_fmt = "%.2f"|format(analysis_meta.get('stability_mean_ari', 1.0)) %} {% set _jac_fmt = "%.2f"|format(analysis_meta.get('stability_consensus_jaccard_mean', 1.0)) %} {{ _('This analysis was re-run %(runs)d times with different random seeds. Mean Adjusted Rand Index between runs: %(ari)s (1.0 = identical, 0.0 = random). Representative-statement Jaccard overlap: %(jac)s.', runs=analysis_meta.get('stability_runs'), ari=_ari_fmt, jac=_jac_fmt) }} {% else %} {{ _('Stability metrics were not computed for this analysis.     {{ _('Limitations.') }} {{ _('Clustering is fit on the same votes used for group summaries, so inferential quantities are exploratory, not confirmatory. Benjamini–Hochberg FDR is appropriate under independence or mild dependence; many tests on the same clustering are correlated, so treat q=0.05 as a practical screen, not a tight bound. Divisive statements compare the two clusters with the highest and lowest agreement on that statement — the Fisher p-value guards the gap in small samples but is not a test of all pairwise cluster differences. The interval on the agreement gap uses Wilson bounds heuristically; a Newcombe interval would be tighter for a difference of proportions.') }}
